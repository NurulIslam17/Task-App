
## About

This is a simple task management app where there are 3 types of users based on their role. These are ADMIN, TEAM_LEAD, and EMPLOYEE.

- ADMIN can perform all actions.
- ADMIN assigns the user a team lead, and the default user will be an employee.
- TEAM LEAD Create and Assign Task
- Employee can see only their task and perform only their action.

## Process

- ADMIN creates a team lead and team and sends an email notification.
- TEAM LEAD Create and Assign Task
- The assigned employee will be notified.



## Worked on

- Java , Spring Boot and Thymleaf
- JWT authentication and authorization
- Spring Securities, Hibernate
- Scheduling, Job-Queue
- Mailtrap service is used for sending email.
- MySQL
- Implemented JPA query for some features
